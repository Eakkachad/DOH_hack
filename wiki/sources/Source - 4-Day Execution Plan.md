# Source: CarbonWay 4-Day Execution Plan (v3)

The master execution plan for the CarbonWay hackathon sprint. A strict 4-day specification covering team setup, JSON schema contracts, role boundaries, daily step-by-step tasks for all 4 team members (P1 Data, P2 Frontend, P3 Pitch, Advisor), risk management, and a submission checklist.

**Key contents:**
- 10 exact formulas (T-VER, VSL, Safety Floor)
- JSON schema for 3 data files (baseline, interventions, scenario_output)
- 10-slide pitch structure with word budgets
- 7-scene video script template (300s, ≤700 words)
- 6 test cases for Safety Floor
- Risk response matrix

**Status:** Finalized. Team uses this as the single source of truth for the 4-day build sprint.
