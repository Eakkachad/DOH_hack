# M-Flow — ระบบเก็บค่าผ่านทางอัตโนมัติแบบไม่มีไม้กั้น

## Overview
**M-Flow** (Multi-Lane Free Flow) คือระบบจัดเก็บค่าผ่านทางอัตโนมัติแบบ **ไม่มีไม้กั้น (Free Flow Tolling)** พัฒนาโดยกองทางหลวงพิเศษระหว่างเมือง กรมทางหลวง เปิดใช้งานครั้งแรกบนทางหลวงพิเศษหมายเลข 9 (M9) เมื่อวันที่ **15 กุมภาพันธ์ 2565** ใช้เทคโนโลยี Automatic License Plate Recognition (ALPR) เพื่อให้รถทุกประเภทสามารถขับผ่านด่านได้โดยไม่ต้องหยุด ชำระเงินทีหลังผ่านช่องทางออนไลน์

## Pain Point
- ความหนาแน่นบริเวณหน้าด่านเพิ่มขึ้นเฉลี่ย **4% ต่อปี**
- ผู้ใช้ทางเสียเวลารอคิว สิ้นเปลืองน้ำมัน และก่อมลภาวะ
- ต้นทุนเศรษฐกิจจากการไม่มี Free Flow: **฿5.97 พันล้าน/ปี** (VOT ฿3.91B + VOC ฿1.53B + Emission ฿523M)

## วิธีการทำงาน
1. **สมาชิก:** ลงทะเบียนผ่าน App/Website/LINE OA, เลือกวิธีชำระเงิน
2. **ขับผ่าน:** ขับรถผ่านด่าน M-Flow — กล้อง ALPR อ่านป้ายทะเบียน
3. **ชำระเงิน:** ได้รับใบแจ้ง e-billing รายครั้ง/รายรอบ ชำระผ่านช่องทางต่าง ๆ
4. **ผู้ที่ไม่ใช่สมาชิก:** ตรวจสอบยอดและชำระผ่าน App/Website ภายในระยะเวลาที่กำหนด

## ผลการประเมิน

| ตัวชี้วัด | ผลลัพธ์ |
|---|---|
| ความเร็วเฉลี่ยที่ผ่านด่าน (ก่อน → หลัง) | เพิ่มขึ้น **189%** (เช่น 12 → 36 km/h) |
| อัตราการไหลของกระแสจราจร | เพิ่มขึ้น **31%** |
| M-Flow throughput | **1,500–2,000 คัน/ชม./ช่อง** (vs ETC 800–1,200, MTC 400) |
| ลดการใช้น้ำมัน (M9) | **13.91 ล้านลิตร/ปี** (มูลค่า ~฿1,200M) |
| ลดเวลาเดินทาง (M9) | **3.33 ล้านชม./ปี** (มูลค่า ~฿919M) |
| ลดมลภาวะ (CO₂, NOₓ, PM) | มูลค่า ~฿93M/ปี |
| NPV ภาพรวม M9 | **฿1,510 ล้าน**, EIRR = 896%, B/C = **6.94 เท่า** |
| สมาชิก (ถึง ก.ค. 66) | **534,591 ราย**, 656,000 คัน |
