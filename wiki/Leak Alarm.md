# Leak Alarm — ชุดแจ้งเตือนไฟฟ้ารั่วที่เสาไฟฟ้าแสงสว่าง

## Overview
**LEAK ALARM** คือชุดอุปกรณ์แจ้งเตือนไฟฟ้ารั่วราคาถูก พัฒนาโดยส่วนไฟฟ้า สำนักงานทางหลวงที่ 12 เมื่อมีไฟฟ้ารั่วที่เสาไฟฟ้าแสงสว่างบนทางหลวง หลอด **LED จะสว่างขึ้น** และระบบจะ**แจ้งเตือนในกลุ่ม LINE** พร้อม Link พิกัดสถานที่ โดยประชาชนสามารถสแกน **QR Code** บนเสาไฟเพื่อแจ้งเหตุได้ทันที

## Pain Point
- ไฟฟ้ารั่วที่เสาไฟฟ้าแสงสว่างทำให้ผู้ไปสัมผัสอันตรายถึงชีวิต
- สาเหตุหลัก: รอยต่อชำรุด, หลุดหลวม, เบรกเกอร์ชำรุด, ปลั๊กชำรุด, บาลาสต์ไหม้
- กรมทางหลวงมีความเสี่ยงถูกฟ้องละเมิดหากเกิดอุบัติเหตุ

## วิธีการทำงาน
```
ไฟฟ้ารั่ว → LED บนกล่อง LEAK ALARM สว่าง
         → ประชาชนสแกน QR Code บนเสาไฟ
         → เลือกประเภทการแจ้งเหตุ → ส่ง
         → แจ้งเตือนกลุ่ม LINE เจ้าหน้าที่ พร้อม Link พิกัด
```

## ต้นทุน & วัสดุ
- **ราคาต่อชุด: 25 บาท** เท่านั้น
- วัสดุ: กล่องพลาสติก, Resistor, Diode, สายไฟ, LED
- ผลิตโดยเจ้าหน้าที่ส่วนไฟฟ้า สทล.12

## ประโยชน์
- ประชาชนได้รับแจ้งเตือนทันที — ปลอดภัยจากไฟฟ้ารั่ว
- ประหยัดงบประมาณและลดเวลาตรวจสอบ
- กรมทางหลวงลดความเสี่ยงถูกฟ้องละเมิด
- แจ้งเหตุง่าย — ประชาชนสแกน QR Code ได้เลย
