# รางวัลดีเยี่ยม — การบริหารงานบำรุงรักษาทางหลวง — โดยระบบสารสนเทศ

## Overview
นวัตกรรมด้านกระบวนงาน (Work Process) ที่ได้รับ **รางวัลดีเยี่ยม** — การจัดระบบงานบำรุงรักษาทางหลวงจากรูปแบบเดิมสู่ระบบสารสนเทศดิจิทัล สอดคล้องตัวชี้วัดนวัตกรรมทางหลวงยุค 4.0

## วิธีการทำงาน
- แปลงกระบวนการบำรุงรักษาจากรูปแบบกระดาษสู่ระบบดิจิทัล
- ตรวจสอบสถานะงานบำรุงทางแบบ Real-time ผ่าน Application
- ใช้ Google Map ในการแสดงพิกัดงาน
- ลดขั้นตอนการรายงาน สะดวก รวดเร็ว

## ผลลัพธ์
- สามารถติดตามผลงานบำรุงทางได้ทุกที่ ทุกเวลา
- สอดคล้องกับ DOH Innovation ยุค 4.0
- ลดเอกสารกระดาษ เพิ่มประสิทธิภาพ
