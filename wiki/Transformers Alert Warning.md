# Transformers Alert Warning — ชุดแจ้งเตือนหม้อแปลงถูกโจรกรรม

## Overview
**TRANSFORMERS ALERT WARNING** คือชุดแจ้งเตือนอัตโนมัติ เมื่อ**หม้อแปลงไฟฟ้าถูกตัดกระแสไฟ** (ซึ่งเป็นขั้นตอนแรกของการขโมย) ระบบจะ**ส่งเสียง Buzzer ดังขึ้น** และ**แจ้งเตือนกลุ่ม LINE** พร้อม Link พิกัดสถานที่ ให้เจ้าหน้าที่ตรวจสอบผ่าน CCTV ได้ทันที พัฒนาโดยส่วนไฟฟ้า สำนักงานทางหลวงที่ 12

## Pain Point
- หม้อแปลงไฟฟ้าถูกโจรกรรมบ่อยครั้ง — ส่งผลให้:
  - ไฟฟ้าแสงสว่างถนนดับ
  - ไฟสัญญาณจราจรใช้การไม่ได้
  - ประชาชนขาดความปลอดภัยและความสะดวก
  - งบประมาณจัดซื้อหม้อแปลงทดแทนสูง

## วิธีการทำงาน
```
หม้อแปลงถูกตัดกระแสไฟ
→ ส่งเสียง Buzzer ดัง (เตือนขโมย/ประชาชนรอบข้าง)
→ แจ้งเตือนกลุ่ม LINE เจ้าหน้าที่ + Link พิกัด
→ เจ้าหน้าที่ตรวจสอบผ่าน CCTV และระงับเหตุ
```

## อุปกรณ์ที่ใช้
- MINI UPS (สำรองไฟ)
- CCTV
- Pocket WiFi + SIM
- Digital Timer
- Buzzer

## ต้นทุน
- **ราคาต่อชุด: 5,000 บาท**
- ผลิตโดยเจ้าหน้าที่ส่วนไฟฟ้า สทล.12

## ประโยชน์
- ประชาชนได้รับความสะดวกและปลอดภัย — ไฟถนนและสัญญาณจราจรทำงานปกติ
- ประหยัดงบประมาณจัดซื้อหม้อแปลงทดแทน
- ยับยั้งการโจรกรรมด้วยเสียงเตือนและการแจ้งเจ้าหน้าที่อย่างรวดเร็ว
