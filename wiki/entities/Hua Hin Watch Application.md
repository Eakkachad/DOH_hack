# Hua Hin Watch Application — แอปบริหารโครงการก่อสร้างทาง

## Overview
**Hua Hin Watch** เป็นแอปพลิเคชันสำหรับ **บริหารและติดตามโครงการก่อสร้างทางหลวง** พัฒนาโดยสำนักงานทางหลวงที่ 15 (ประจวบคีรีขันธ์) ต่อยอดจากระบบ PLANNET2021 โดยนำข้อมูลมาแสดงในรูปแบบที่ผู้บริหารเข้าใจง่าย ผ่านกราฟ แผนภูมิ และตาราง สร้างด้วย **Glide App** (ฟรี) บนฐานข้อมูล Spreadsheet

## Pain Point
- งานก่อสร้างจำนวนมาก ต้องจัดการเอกสารและรายงานจำนวนมาก
- การติดตามความคืบหน้าทำได้ยาก รูปแบบการรายงานไม่เป็นมาตรฐาน
- ผู้บริหารไม่สามารถเห็นภาพรวมได้อย่างรวดเร็ว

## คุณสมบัติหลัก (13 หมวดหมู่)

| โมดูล | เนื้อหา |
|---|---|
| แผนงาน | แผนงานทั้งหมดของ สทล.15 และสถานะ |
| แบบโครงการก่อสร้าง | แบบก่อสร้าง สถานะการจัดทำ ไฟล์แบบ |
| จัดซื้อจัดจ้าง | กระบวนการจัดซื้อและสถานะ |
| ส่วนควบคุมงานก่อสร้าง | บริหารสัญญา, แก้ไขแบบ, แก้ไขสัญญา, ถัวจ่าย |
| ขยายอายุสัญญา/งด-ลดค่าปรับ | กราฟและรายละเอียดโครงการ |
| การเบิกจ่าย | ภาพรวมการเบิกจ่ายงบประมาณทุกหน่วย |
| สรุปรายงาน | Dashboard ภาพรวมและลิงก์สู่รายละเอียด |
| ถาม/ตอบ | แผนที่, ช่องทางติดต่อ, แบบฟอร์มส่งคำถาม |

## เทคโนโลยีที่ใช้
- **Platform:** Glide App (No-code, ฟรี)
- **ฐานข้อมูล:** Google Spreadsheet (ดึงข้อมูลจาก PLANNET2021)
- **UI:** กราฟ, แผนภูมิ, ตาราง — ออกแบบให้เข้าใจง่าย

## เป้าหมาย
- จำนวนโครงการและงบประมาณที่กันเหลื่อมปี 2568 น้อยกว่าปีที่ผ่านมา
- ผลการเบิกจ่ายตามเป้าหมาย ดีกว่าปีที่ผ่านมา
- เพิ่มประสิทธิภาพการบริหารสัญญา
