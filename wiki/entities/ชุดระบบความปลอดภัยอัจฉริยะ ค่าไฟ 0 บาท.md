# ชุดระบบความปลอดภัยอัจฉริยะ ค่าไฟ 0 บาท

## Overview
**ชุดระบบความปลอดภัยอัจฉริยะ** (Intelligent Safety System) คือนวัตกรรม **3-in-1** ที่คิดค้นโดยแขวงทางหลวงบึงกาฬ ประกอบด้วย 3 ระบบในชุดเดียว:

| ระบบ | รายละเอียด |
|---|---|
| 1. กล้องวงจรปิด (CCTV) | เฝ้าระวังพื้นที่ 24 ชม. |
| 2. ชุดควบคุมไฟโซล่าเซลล์ | จ่ายไฟด้วยพลังงานแสงอาทิตย์ — **ค่าไฟ 0 บาท** |
| 3. สัญญาณไฟกระพริบ + ป้ายเตือน | เพิ่มความปลอดภัยบริเวณจุดเสี่ยง |

มี 2 แบบ: **แบบที่ 1 — ฝังเสา** (ติดตั้งถาวร) และ **แบบที่ 2 — เคลื่อนที่ได้**

## Pain Point & แรงบันดาลใจ
- การจลกรรมทรัพย์สิน (ขโมยอุปกรณ์ทางหลวง)
- นโยบายผู้บริหาร — ต้องการระบบรักษาความปลอดภัยที่ครอบคลุมมากขึ้น
- ต้องการลดค่าไฟฟ้าจากการติดตั้งอุปกรณ์ความปลอดภัย

## กระบวนการพัฒนา
1. ศึกษาเรียนรู้ข้อมูลเพิ่มเติม + สำรวจวัสดุเหลือใช้จากสำนักงาน
2. วางแผนการทำงานให้อยู่ในงบประมาณที่กำหนด
3. ออกแบบโครงสร้าง + สั่งซื้ออุปกรณ์บางชิ้นที่ขาด
4. ประกอบและติดตั้ง
- **ระยะเวลาดำเนินการ: 7–10 วัน**

## หน่วยงานที่ใช้
- แขวงทางหลวงบึงกาฬ
- หมวดทางหลวงในสังกัดแขวงทางหลวงบึงกาฬ
- ศูนย์อำนวยความสะดวกช่วงเทศกาล

## ผลลัพธ์
- ของหายน้อยลง — ลดการสูญหายของทรัพย์สิน
- นำวัสดุเหลือใช้มาใช้ประโยชน์ (Upcycling)
- ได้ระบบรักษาความปลอดภัยที่ครอบคลุมมากขึ้น
- **ค่าไฟฟ้า 0 บาท** — ใช้พลังงานแสงอาทิตย์
