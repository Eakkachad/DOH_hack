# Project Management — ระบบบริหารโครงการก่อสร้างทางดิจิทัล

## Overview
ระบบ **Project Management** พัฒนาโดย ศูนย์สร้างทางสงขลา [[กรมทางหลวง]] โดยนายชาชิน พวงประทีป หัวหน้าฝ่ายปฏิบัติการ เพื่อติดตามความก้าวหน้าโครงการก่อสร้างทางดิจิทัล ผ่านระบบ Progress Chart และ Workdone แบบ Real-time

## Pain Point
- การติดตามความก้าวหน้าโครงการก่อสร้างยังเป็นแบบเดิม (Manual)
- ขาดระบบดิจิทัลที่แสดงผล Progress แบบ Real-time
- ข้อผิดพลาดในการรายงานและติดตามงานก่อสร้าง

## วิธีการทำงาน
1. **Progress Chart** — ติดตามแผนงานและ Workdone แบบอัปเดตอัตโนมัติ
2. **ระบบดิจิทัล** — ลดข้อผิดพลาดในการบริหารงานก่อสร้าง
3. **ควบคุมแผน** — ติดตามความคืบหน้าของโครงการเทียบกับแผน

## ผลลัพธ์
- ลดข้อผิดพลาดในการรายงานงาน
- ติดตามความก้าวหน้าโครงการได้แบบ Real-time
- ควบคุมคุณภาพงานก่อสร้างตามแผนที่วางไว้
