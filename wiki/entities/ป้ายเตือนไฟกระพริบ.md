# ป้ายเตือนไฟกระพริบ — Solar LED Warning Sign จากวัสดุเหลือใช้

## Overview
**ป้ายเตือนไฟกระพริบ** คือป้ายเตือนความปลอดภัยบนทางหลวง พัฒนาโดยแขวงทางหลวงธนบุรี โดยนำ**เศษวัสดุเหลือใช้**จากงานทางหลวงมาประกอบเป็นป้าย LED กระพริบ ขับเคลื่อนด้วย**แผง Solar Cell** — ไม่ต้องต่อไฟฟ้า ใช้พลังงานแสงอาทิตย์ชาร์จแบตเตอรี่ เพื่อเพิ่มทัศนวิสัยในการมองเห็นของผู้ใช้ถนนในยามค่ำคืน

## Pain Point
- สถิติอุบัติเหตุบนทางหลวงส่วนใหญ่เกี่ยวข้องกับ**อันตรายข้างทาง** และการเฉี่ยวชนอุปกรณ์ข้างทาง
- ผู้ใช้ถนนขาดความรู้ด้านการจราจรและอุปกรณ์ความปลอดภัย
- แขวงทางหลวงธนบุรีมีเศษวัสดุเหลือใช้จากงานทางจำนวนมาก

## วัสดุที่ใช้ (เน้นวัสดุเหลือใช้)

| วัสดุ | รายละเอียด |
|---|---|
| แผ่นป้ายอลูมิเนียม | 60 × 1.20 เมตร จำนวน 1 แผ่น **(เหลือใช้)** |
| เหล็กกล่อง 2×1 นิ้ว | 8.6 เมตร **(เหลือใช้)** |
| แผง Solar Cell | 30W จำนวน 1 แผง **(เหลือใช้)** |
| Battery 12V/12Ah | 1 ลูก **(เหลือใช้)** |
| LED Flash Light | 4 ชุด **(เหลือใช้)** |
| ตู้พลาสติกกันน้ำ | 1 กล่อง |
| ล้อเลื่อน | 1 ชุด (เคลื่อนย้ายได้) |

**ต้นทุนรวม: 2,605 บาท** (วัสดุที่ต้องจัดหาเพิ่มเติมเพียงบางส่วน)

## การทำงาน
```
กลางวัน: แผง Solar Cell ผลิตไฟ → ชาร์จแบตเตอรี่
กลางคืน/เมฆครึ้ม: แบตเตอรี่จ่ายไฟให้ LED กระพริบ
→ เพิ่มทัศนวิสัย เตือนผู้ใช้ถนน
→ ปลอดภัยสำหรับผู้ปฏิบัติงานและผู้ใช้ทาง
```

## ประโยชน์
- **ความปลอดภัย:** ลดอุบัติเหตุข้างทาง เพิ่มทัศนวิสัยกลางคืน
- **ประหยัดงบ:** ใช้วัสดุเหลือใช้, แบตชาร์จซ้ำได้ ไม่มีค่าไฟ
- **สิ่งแวดล้อม:** ลดขยะ, ใช้พลังงานแสงอาทิตย์
- **ยืดหยุ่น:** มีล้อเลื่อน เคลื่อนย้ายไปใช้งานได้ตามจุดต่าง ๆ
- **ต่อยอดได้:** สามารถนำไปประยุกต์ใช้ในรูปแบบอื่น
