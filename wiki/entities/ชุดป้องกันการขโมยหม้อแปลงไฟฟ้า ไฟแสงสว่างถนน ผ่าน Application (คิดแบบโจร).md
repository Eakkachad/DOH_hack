# ชุดป้องกันการขโมยหม้อแปลงไฟฟ้า ผ่าน Application (คิดแบบโจร)

## Overview
**ชุดป้องกันการขโมยหม้อแปลงไฟฟ้า ไฟแสงสว่างถนน ผ่าน Application Line** คือระบบตรวจจับและแจ้งเตือนอัตโนมัติ พัฒนาโดยแขวงทางหลวงอุบลราชธานีที่ 1 ด้วยแนวคิด **"คิดแบบโจร"** — ออกแบบระบบโดยคิดในมุมมองของผู้ขโมย เพื่อออกแบบการตรวจจับที่แม่นยำ เมื่อไฟฟ้าถูกตัด ระบบจะ:
1. ส่งเสียง **ไซเรน + แตร** ดังขึ้นทันที
2. ส่งแจ้งเตือน **LINE พร้อมพิกัดสถานที่** ให้เจ้าหน้าที่ตลอด 24 ชม.
3. มีระบบสำรองไฟ (UPS) เพื่อให้ยังส่งสัญญาณได้แม้ไฟหลักถูกตัด

## Pain Point
- แขวงทางหลวงอุบลราชธานีที่ 1 มีเสาไฟฟ้าแสงสว่างจำนวนมาก
- หม้อแปลงไฟฟ้าราคาสูง — ถูกลักขโมยบ่อยครั้ง
- ส่งผลต่อความปลอดภัยของประชาชน (ไฟดับ) และสูญเสียงบประมาณ

## สถาปัตยกรรมระบบ

```
ฟิวส์แรงสูง/แรงต่ำถูกปลด
→ เซ็นเซอร์ตรวจจับแรงดันไฟ (AC Voltage Sensor)
→ Arduino ESP8266 ประมวลผล
  → ไซเรน + แตร ดังทันที (ยับยั้งขโมย)
  → Router WiFi ใส่ซิม ส่ง LINE Notification + พิกัด GPS
  → UPS + Battery สำรองไฟ (ระบบยังทำงานแม้ไฟดับ)
```

## ประโยชน์
- **ตรวจจับ 24 ชม.** — แจ้งเตือน LINE ทันทีพร้อมพิกัด
- **ลดความเสียหาย** — เจ้าหน้าที่รับทราบและระงับเหตุได้เร็ว
- **ประชาชนปลอดภัย** — ไฟแสงสว่างถนนไม่ดับ
- **ลดงบประมาณ** — ไม่ต้องซื้อหม้อแปลงทดแทนบ่อย
- **ต่อยอดได้** — พัฒนา Line Application ให้มีประสิทธิภาพยิ่งขึ้น
