# หลักเกณฑ์การขอใช้บริการคลาวน์ภาครัฐ GDCC (ก.ย. 2564)

## Overview
หลักเกณฑ์การขอใช้บริการ **Government Data Center and Cloud Service (GDCC)** สำหรับหน่วยงานภาครัฐ รวมถึง[[กรมทางหลวง]] กำหนดแนวทางการใช้ระบบ Cloud ภาครัฐ
