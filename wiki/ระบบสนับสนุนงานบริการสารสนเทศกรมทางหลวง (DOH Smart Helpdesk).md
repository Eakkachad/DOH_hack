# DOH Smart Helpdesk — ระบบสนับสนุนงานบริการสารสนเทศกรมทางหลวง

## Overview
**DOH Smart Helpdesk** (https://smhd.doh.go.th) คือระบบดิจิทัลสำหรับรับแจ้งและติดตามแก้ไขปัญหาด้านสารสนเทศของ[[กรมทางหลวง]] ครอบคลุม Hardware, Software, Network และระบบงานต่างๆ ทั่วประเทศ

## Pain Point
- หน่วยงานทั่วประเทศมีปัญหาด้าน IT หลากหลาย (Hardware/Software/Network)
- ขาดระบบรวมศูนย์ในการติดตามสถานะการแก้ไข
- กระบวนการแจ้งปัญหาเดิมเป็นแบบ Manual ล่าช้า

## วิธีการทำงาน (SDLC)
ระบบพัฒนาตามมาตรฐาน Software Development Life Cycle (SDLC) 6 ขั้นตอน:
1. **วางแผน** — รวบรวมความต้องการ
2. **ออกแบบระบบ** — System Flow Diagram, Use Case Diagram
3. **พัฒนาระบบ** — เขียนโปรแกรม
4. **ทดสอบ** — ตรวจสอบความถูกต้อง
5. **ติดตั้งระบบ** — Deploy ใช้งานจริง
6. **บำรุงรักษาระบบ** — ดูแลต่อเนื่อง

### ฟีเจอร์หลัก
- รับแจ้งปัญหา IT ออนไลน์ (Hardware, Software, Network)
- ติดตามสถานะการดำเนินการแบบ Real-time
- รายงานสรุปสถิติปัญหา IT ทั้งองค์กร
- ระยะเวลาดำเนินการ 10 เดือน
