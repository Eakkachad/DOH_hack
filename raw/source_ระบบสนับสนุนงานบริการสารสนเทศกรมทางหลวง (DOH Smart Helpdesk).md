## Summary
ITC ：DOH
ระบบสนับสนุนงานบริการสารสนเทศ
รับแจ้งติดตามและแก้ไขปัญหาด้านสารสนเทศเช่น Hardware Software
Network และระบบงานที่กรมทางหลวงให้บริการอยู่
https://smhd.doh.go.th

วัตถุประสงค์
1. เพื่อปรับเปลี่ยนกระบวนการปฏิบัติงานของ
เจ้าหน้าที่กรมทางหลวงโดยนาเทคโนโลยีดิจิทัล
เข้ามาประยุกต์ใช้งาน
2. เพื่อให้มีระบบสนับสนุนงานบริการสารสนเทศ
กรมทางหลวงในกระบวนการแจ้งปัญหาติดตาม
สถานะและประเมินผลได้อย่างสะดวกรวดเร็ว
เป้าหมาย
1.
กรมทางหลวงก้าวเข้าสู่การเป็นองค์กรดิจิทัล
2. ลดการหยุดชะงักในการปฏิบัติงานและการ
ให้บริการประชาชนของเจ้าหน้าที่กรมทางหลวง
3. ผู้ใช้งานสามารถติดตามสถานะของการแจ้ง
ปัญหาด้านเทคโนโลยีดิจิทัลของตนเองได้
ตลอดเวลา
4. การปฏิบัติงานของเจ้าหน้าที่ผู้รับผิดชอบ 
มีประสิทธิภาพยิ่งขึ้นจากผลการประเมิน 
ความพึงพอใจ
